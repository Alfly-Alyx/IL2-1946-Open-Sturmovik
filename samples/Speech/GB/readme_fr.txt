
I. Installation
---------------

Cet ensemble de fichiers est destiné à prendre place dans un sous-répertoire dont le chemin sera (à partir du dossier d'installation de "Forgotten Battles") SAMPLES/SPEECH/FR. La procédure d'installation est donc la suivante :

1. Créer un répertoire FR dans le sous-dossier SAMPLES/SPEECH (il prendra place à côté des répertoires DE, FI, HU, RU et US)
2. Décompresser les fichiers de l'archive dans ce nouveau dossier (il faudra veiller à ce que l'option "conserver les chemins" de votre logiciel de décompression soit activée)

Vous devriez, à ce stade, avoir un certain nombre de sous-dossiers créés dans le dossier FR : Actor1, Actor2, ..., Actor9. Ces dossiers contenant, eux-mêmes, de nombreux fichiers.

II. Utilisation "normale" de ces fichiers
-----------------------------------------

Cet ensemble de voix est destiné à introduire des voix françaises dans Il2 "Forgotten Battles". En fait, il est conçu avec l'idée de donner la voix des pilotes du groupe de chasse Normandie (qui deviendra plus tard le Normandie-Niemen). Ainsi, les indicatifs sont "Rayack", et les messages débutent par "Allo", car ce sont les termes utilisés, à l'époque, par ces pilotes.

De plus, nous avons des voix pour les pilotes de bombardiers. Pour ceux-ci nous avons utilisé l'indicatif Bretagne. En effet, c'est le groupe de bombardement qui fut préssenti pour rejoindre le Normandie en URSS. Néanmoins, ce groupe ne fut finalement pas envoyé en Russie.

Pour entendre ces voix "françaises", il faut que la mission que vous effectuez comporte des pilotes "français". Ainsi, dans l'éditeur de mission, il faut attribuer la nationalité française à certains groupes de vol. C'est le cas si vous attribuer l'escadrille Normandie-Niemen à ces groupes.

De plus, dans ce cas, les dialogues entre la tour et les pilotes français seront, eux aussi, en français. Ce qui est d'ailleurs parfaitement historique !

III. Utilisation possible de ces fichiers
-----------------------------------------

Il est possible de remplacer toutes les voix du jeu par les voix de cet additif. Il suffit, pour cela, de remplacer tous les fichiers de tous les répertoires du dossier SAMPLES/SPEECH.

Ainsi, si l'on désire que les unités allemandes utilisent ces fichiers, plutôt que les voix allemandes, il suffit de placer nos fichiers dans le dossier DE.

Bien entendu, on y perdra en réalisme. Néanmoins, on pourra, ainsi, se passer des textes de traduction qui encombrent l'écran.

IV. Notes des concepteurs
-------------------------

Les voix utilisées dans ce travail sont dotées de divers accents. Nous avons utilisé plusieurs personnes pour que l'on ait un large éventail des accents français. 

Nous avons aussi modifié la structure des messages. Ainsi, certains messages sont "vides". Alors que les éléments qu'ils fournissaient sont incorporés dans les autres messages. Nous pensons avoir, ainsi, donner plus de punch à l'ensemble. Certains messages sont un peu différents de ceux de base, pour mieux coller à la situation qu'ils décrivent et donner un meilleur réalisme. C'est ainsi que certains messages ne correnspondent pas à 100% à leur traduction. Néanmoins, ils collent parfaitement à leur utilisation.

Comme on l'a dit, les messages reprennent les indicatifs historiques du Normandie. On aurait pu, avec une structure plus souple pour cet ensemble de fichiers, effectuer une plus grande différentiation entre les patrouilles et les escadrilles. Nous avons été obligés de nous restreindre un peu...

Nous n'avons pas utilisé les codes "numériques" des VVS (forces arérienne soviétiques), bien qu'ils fussent largement décrits dans les conversations radio du Normandie.
