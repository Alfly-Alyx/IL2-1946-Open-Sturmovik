Overview

"Mission Mate" is a utility program written to both build "Quick Mission" styled missions for "IL-2 Sturmovik 1946" and also to edit  existing single player missions generated by the "Full Mission Builder" or  by the "Mission Mate Quick Mission Builder."

 

One of the key features of the "Mission Mate Quick Mission Builder", is that the user is able to select his/her desired start and end waypoints on the map to fly the quick mission and is not restricted to game play in one location on each map. This feature vastly improves the variety and enjoyment the user can achieve from what is currently offered with the "Forgotten Battles Quick Mission" . This feature is explained in detail in the "Quick Mission Builder" Section. Beyond this Mission Mate also offers a variety of multiple waypoint mission scenarios that will enable you to very quickly set up exciting and varied missions. You will be limited only by your imagination.

 

"Mission Mate" runs independently from "IL-2 Sturmovik 1946" and therefore should be launched before starting "IL-2 Sturmovik 1946" itself. Once Mission Mate and IL-2 Sturmovik 1946 programs  are running simply use the Windows standard key strokes "Alt" + "Tab" to toggle between the two applications. 

 

"Mission Mate" has been developed using Visual Basic 6.0 and tested with Windows XP home and Professional editions. It has been designed for single player styled missions and game play. There is however, an option to save any mission as a co-op style mission through the "Mission Editor" interface.

Quick Mission Builder

 

Air To Air (Back To Subject Index)  (Back To Window)

Related to  Quick Mission Builder

    This is the primary method of mission building. You may choose to have either unassigned or assigned attacks for aircraft, airbase and related objects and anchored shipping. Mission scenarios might include dog fighting, bomber attack, bomber escort, ground attack and strafing.

The other two modes of mission generation are vehicle attack and shipping attack. These two modes are assigned attacks, meaning that all flight groups in the army chosen for the attack are assigned to that task specifically as their main objective, whereas in the air to air mode individual flight groups may be carrying out different tasks.

Double Variable Airbase Occupation  (Back To Subject Index)  (Back To Window)

Related to  Quick Mission Builder

    Some maps  incorporate a feature that allows the user to change which army may occupy either base at any time (Kyushu, New Guinea and Okinawa). To switch from one army to another just hold the mouse pointer over the airbase icon and scroll the mouse wheel, this will toggle between red and blue armies for that base. This means it is possible to have both bases occupied by the same army at one time, this however then requires the existence of carrier points on the map so the opposing army has a landing and takeoff point. Landing and takeoff points are automatically assigned as you toggle airbase occupation.

When both bases are occupied by the same army you may toggle landing and take off points independently, by using a combination of "shift" + mouse scroll wheel for landing and "ctrl" + mouse scroll wheel for take off. If both landing and take off points are at the same base both may be toggled together by using "shift" + "ctrl" + mouse scroll wheel.

 

Setting airbase options for these maps is also different from the other maps and is invoked by holding the mouse pointer over the airbase icon and clicking the mouse wheel as a button. This will invoke the "Airbase Options" window as shown below.

Multiple Waypoints (Back To Subject Index)  (Back To Window)

Related to Quick Mission Builder

   Please note: The principals explained in this section for moving, adding and deleting multiple waypoints also apply to naval waypoints.

 

    This mode offers the greatest flexibility to the quick mission scenarios and offers the most complete and meaningful feel to quick mission game play. This is because the flight groups of each team may have different objectives defined by multiple waypoints. The attack point is user defined from the waypoints on the map interface and each waypoint may be edited for both altitude and speed via a pop up menu over the desired waypoint from the map interface. Both the speed and altitude of points placed on the map are governed by the currently active flight group spacing and speed method when placing waypoints. 

For example, if you are using default settings and are placing player waypoints, the altitude listed in the "Start Altitude" drop down list in the "Player Flight Groups Frame" will determine the altitude of each waypoint placed as will the altitude difference between flight group 1 and flight 4 by the "Grp 4 Above Lead"  drop down list determine the altitude variance between  flight level 1 and flight level 2. So if you changed the altitude setting in the "Start Altitude" drop down list for each waypoint while placing points, you would have different altitudes for each of the waypoints placed, as you could also do for altitude variance although this will probably not be desired. If however you had the "Advanced Flight Group Settings" check box selected, the settings inside the "Advanced Options" window would set the values for speed and altitude variance between flight level 1 and flight level 2.

 

    To place multiple player flight group waypoints, first select the "Multiple Waypoints" option. Using the left mouse button start placing your desired points on the map by clicking one point then the next until you have placed as many waypoints as desired. To stop the waypoint placement process, hold down the "Shift" key on your keyboard and click the left mouse button again on the map or press the "Esc" key. This will cause Mission Mate to stop the placement process.

    To place multiple AI flight group waypoints, follow the same steps as above but use the right mouse button to place points. To end the process, hold down the "Shift" key on your keyboard and click the right mouse button again on the map or press the "Esc" key. This will cause Mission Mate to stop the placement process for AI points.

NOTE: When placing waypoints on the map it is advisable to avoid creating sharp acute angles between waypoints. This is because Mission Mate calculates parallel flight paths for the different flight groups for each team, so if the angles are too sharp, the outer flight groups will have a far greater distance to fly to their waypoint compared to the inner flight groups. See examples below.

 

Wider flight grouping is also recommended for multiple waypoints as the individual planes within each group tend to swing out and disperse slightly at each turn/waypoint.

NOTE: When placing waypoints on the map it is advisable to avoid creating sharp acute angles between waypoints. Apply New Mod Aircraft  (Back To Subject Index)  (Back To Window)

Related to  Quick Mission Builder

    Clicking the apply button activates several actions. Firstly, Mission Mate examines the contents of the ru_properties (the file which holds all the load-out information for each aircraft) and extracts load-out information for each aircraft in the user defined list, providing a matching entry is found. Next Mission Mate generates a loud-out file based on this information and writes it to the Mission Mate "User Defined Loadouts" folder located at the root of the Mission Mate program folder. An entry for each aircraft is produced. If Mission Mate cannot find a matching aircraft entry in the ru_properties, a default load file is created. The next step is to create a blank skin folder for the aircraft based on the information found in the air.ini file as indicated in both lists. If there is already a skin folder present with the same name Mission Mate will bypass this operation and leave the existing folder intact. Finally Mission Mate generates an entry for each aircraft code into a file named "MMModFlyable.txt" which can be found at the root of the Mission Mate program folder. This file holds the entries for each mod aircraft added by you including the AI flyable entries. These entries may also be edited manually by you.
