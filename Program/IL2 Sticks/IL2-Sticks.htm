IL2-Sticks, version 1.0h Beta!

This tool will allow you to:

  • Configure the sensitivity settings on your Multi-Controller IL2 Setup, per defined Pil2ot, using sliders, much like IL2 itself can on the first controller.
  • Save Sensitivity Profiles as stored in Conf.ini into separate files, and assigning them back to Conf.ini. Thus you can create profiles for either different planes, or different Pilots, or to your likings … or tests for that matter.

This tool was created and tested for IL2 Forgotten Battles, merged with the Ace Expansion Pack, ánd Pacific Fighters, patched up to 4.01m. It may work on versions as of IL2 Forgotten Battles, and / or Pacific Fighters stand alone. It will also work for IL2 Sturmovik (The Original) from versions 1.04 and up, though only FB+AEP+PF 4.01m have seen testing!

 

Installing

Make sure a folder called ‘Profiles’ exists as a subfolder to the folder where IL2-Sticks.exe is located.

 

Initial Startup

At initial startup, this tool will assume the IL2 game is located in folder ‘C:\Program Files\Ubi Soft\IL-2 Sturmovik Forgotten Battles’.

If yours is not, it will display a message, and present a box where you can manually enter your game folder.

It will also assume ‘Profiles’ is a subfolder to the program-folder. You can change that location, by manually editing the IL2-Sticks.ini file.

 

How does it work?

Il2-Sticks will check the IL2 program folder for all in-game defined pilots. It will list them in a listbox called ‘Pilot’. (The currently in-game selected pilot will be initially selected)

Next, it will read the pilot’s settings.ini file, and it will select all in-game defined controllers from the HOTAS section in Control.

Then, it will read Conf.ini and retrieve the sensitivity settings for the controllers in HOTAS.

These settings are show in 10 sliders for sensitivity, and 2 for DeadBand and Filtering. Much like the in-game screen.

 

Operating IL2-Sticks

  • Make sure you only use this tool, whenever IL2 is not running.
  • Make sure your HOTAS controller assignments in IL2 are set correctly, for all pilots!
  • Select the controller you want to change in the Hotas: list box. You can edit the settings by moving the sliders up or down.
  • Save the new settings: Press Button Save to Conf.ini  and (after confirmation) it will store the new settings to conf.ini, saving the previous version of conf.ini in a file named conf.ini.org. (Note: Whenever you save to conf.ini, any existing file named conf.ini.org will first be deleted, after saving it will contain the old conf.ini contents)
  • Profiles: You can use the button Save Profile to save the current sensitivity settings (stored in conf.ini) into a separate file, which will be placed in the ‘Profiles’ folder. Default file name will be ‘New Profile.ini’ but you can change this to your liking. But, you cannot not use the name ‘Conf.ini’. For your convenience, you can type a short text in the Remark text box, this will be stored in the Profile. However, It is not possible to save this remark to conf.ini!
  • The button Set this Pilot as current in-game will the change the currently in-game selected Pilot to the one selected in ‘Pilot’. (It’s not enabled when the current Pilot is the in-game selected one)
  • Buttons:
    1. All 100 Sets all sliders of currently selected HOTAS to value of 100.
    2. All 0 Sets the sliders to value of 0.
    3. Linear Sets the sliders according to a line, starting 10, ending 100.
    4. Curve Sets the sliders to a parabolic curve, starting 0, ending 100.
    5. Up 10% Moves the sliders up by 10% of their current value.
    6. Down 10% Moves the sliders down by 10% of their current value.
    7. Copy Copies current slider-values to a ClipBoard (It’s not the Windows one, so don’t use Ctrl-C, Ctrl-V)
    8. Paste Pastes sliders-values from that ClipBoard to currently selected sliders.

 

Best Practice:

  1. Save your current Conf.ini sensitivity settings for your main pilot to a (new) profile.
  2. Change the sensitivity settings, save them to a (new) profile, then save them to Conf.ini
  3. Test the new settings, and change them as you please, save them to Conf.ini.
  4. When you think the settings are to your liking, save them to a new profile.
  5. When saving a profile, name it properly, and use the ‘Remark’-field  for future reference!

 

Tips

You can save Profiles either as a Pil2ot’s preference (Oleg’s and Captain Brown’s settings are initially present as profiles!) ór as a plane’s preference.

 

Features present, not by design!

  • By starting IL2-Sticks twice and operating the two (or more!) side by side, you can compare settings! However, the tool will not be aware of this! Saving settings in the first instance will not be known to the other instance of IL2-Sticks. So be carefull in using it this way!

 

Known Issues

  • Only senstivity settings for a controller-Axis assigned to a HOTAS function in-game are shown and saved. Be aware of this when changing pilots and exchanging profiles between them. If used in an incorrect way, sensitivity settings can be lost!
